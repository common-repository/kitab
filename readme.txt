Kitab - Books Management System for WordPress

== Description ==
Kitab (Arabic - كِتَاب) is a powerful plugin that adds all the necessary features to run a book website on WordPress. It adds custom post types for books, as well as custom taxonomies for authors and publishers. You can also add custom fields for books, such as ISBN, number of pages, and edition. You have full control over which custom fields to display and how to display them, with the option to choose between dl and table formats.

== Features ==
- Custom post type for books. [todo]
- Custom taxonomies for authors and publishers. [todo]
- Custom fields for books, such as ISBN, number of pages, and edition.
- Full control over which custom fields to display and how to display them.
- Option to choose between dl and table formats for displaying book info.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/kitab` directory, or search for "kitab" and install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Kitab Menu to configure the plugin.
